# QTodo

Lightweight todo list app made with Qt 5
Calls a rest api for authentication and data storage

## How to build ?

```bash
git clone https://github.com/Ola-jed/QTodo
cd QTodo
cmake .
make
./QTodo
```

Just import the project if you use an ide.
